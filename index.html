Preguntas frecuentes

Esto ayuda mucho cuando lo muestres a un cliente.

Cliente
¿Estos demos ya se pueden usar para un negocio real?

Sí: se pueden personalizar con logo, colores, textos, ciudad y servicios. Si se requiere formulario real (sin mailto), se integra backend o servicios externos.

¿Se puede conectar con WhatsApp y correo del cliente?

Claro. Se ajusta el número con formato wa.me y el correo con mailto. También se puede agregar botón de llamada, redes sociales y Google Maps.

¿Se puede poner dominio propio y SEO?

Sí. Se puede conectar un dominio, optimizar títulos/descripciones, y agregar analítica. Para SEO avanzado se recomienda un sitio multipágina o framework.
